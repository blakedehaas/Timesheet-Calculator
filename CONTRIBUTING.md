# Contributing to Timesheet Calculator

Thank you for considering contributing to this project! Follow these steps to contribute effectively.

---

## 1. Fork the Repository

Click the **Fork** button on GitHub to create a copy of this repository under your account.

---

## 2. Clone the Repository

After forking, clone your copy of the repository to your local machine:

git clone git@github.com:blakedehaas/Timesheet-Calculator.git
cd Timesheet-Calculator

## 3. Create a Feature Branch
Before making any changes, create a new branch to work on:

git checkout -b feature-branch-name

Replace feature-branch-name with a short, descriptive name for your changes.

## 4. Make Changes and Commit
Make your improvements to the code. After making changes, stage and commit them:

git add .
git commit -m "Describe your changes"

Use a clear and concise commit message.

## 5. Push Changes and Create a Pull Request
Push your branch to GitHub:

git push origin feature-branch-name

Then, go to the original Timesheet-Calculator repository on GitHub and:

1. Click Pull Requests.
2. Click New Pull Request.
3. Select your forked repository and branch.
4. Provide a clear description of the changes you made.
5. Click Create Pull Request.

## 6. Wait for Review
A maintainer will review your pull request.
If changes are requested, make the edits and push them to the same branch.
Once approved, your changes will be merged.

If you would like to be a maintainer, reach out!

# Additional Contribution Guidelines
Follow the coding style used in the project.
Keep your changes focused and well-documented.
If submitting a bug fix, mention the issue number in the pull request.
For new features, discuss in GitHub Issues before implementation.
